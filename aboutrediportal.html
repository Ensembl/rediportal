REDIportal Track Hub

Content

The following files contain analysis derived from the REDIportal. Files are formatted as bed6+1 format with the last column being the REDIportal string indicating if a given site resides in ALU elements or REP repetitive not Alu elements or NONREP non repetitive elements. The score is always equal to zero because there are no prediction scores per site. The name field includes now the unique accession number. Each site has an accession independently from the assembly version.
